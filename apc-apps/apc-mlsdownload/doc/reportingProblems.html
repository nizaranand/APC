7.0 Reporting Problems
You should consult the Frequently Asked Question (FAQ) and Sample Configurations sections before reporting problems.

To report problems please post them to the development list. You can add youself to the development list at the following location:

http://mail.crt.realtors.org/mailman/listinfo/vielerets-dev

Posting to the lest helps the entire community. For other issues, please contact:

info@crt.realtors.org

Indicate the version of the software you are using, the operating system you are using and include a short description of the problem. You can look up the version of the package from the About item on the Adminstrative Interface menu.
