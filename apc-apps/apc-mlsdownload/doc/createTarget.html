5.0 Create a Target
After a data source is configured, you can then setup a Target. Target definition is independent of Source definition and you can have many Targets for a single Source.

To configure the site, type the following URL for the Administrative Interface into your browser: http://{SITE_URL}/setup From here, you should see a menu that includes an option to create a Target. You will be asked to assign an TARGET_NAME to each Target. Creating a new Target will create a file at {INSTALLATION_DIRECTORY} / {TARGETS_DIRECTORY} . This directory contains multiple files, one for each Target.

The Target Configuration (Configuration Guide) presents detailed information on Target configuration.
