2.0 Download Distribution
Download the DISTRIBUTION_FILE file from the project website with your browser and store it locally in a convienient directory. This location will be called your DISTRIBUTION_DIRECTORY throughout the rest of this document.

If you decided to store the downloaded file in the directory: C:\MySite then the DISTRIBUTION_DIRECTORY would be called: C:\MySite The DISTRIBUTION_FILE has the following naming scheme: vieleRETS-{VERSION}.{ARCHIVE_TYPE} Where VERSION is the version number and ARCHIVE_TYPE is either ".zip" or ".tar.gz". The following are examples of the naming scheme.
  • vieleRETS-1.1.tar.gz
  • vieleRETS-1.1.zip
  • vieleRETS-1.2.tar.gz
  • vieleRETS-1.2.zip
